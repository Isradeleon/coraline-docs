Getting started

This is and open source CSS only framework. Its main objective is to be a fresh alternative to build high quality websites on the market.

Quick start

You may use the CDN for a quick start:

  • https://cdn.statically.io/gh/isradeleon/coraline/0.5.3/coraline-v0.5.3/coraline.min.css

Easy installation

You may install the framework from npm or download the current version here. Then just import the minified CSS file into your project. Here is the folder structure:

coralinecss
  • coraline-v0.5.3
    • coraline.min.css
  • base.scss
  • base.css
  • README.md
  • LICENSE

The coraline-v0.5.3/coraline.min.css file is production ready.


NPM

Just do npm install coralinecss from your terminal. After installing, you'll find the coraline-v0.5.3 folder with the production minified file. Outside this folder, you'll find the base.scss and base.css development files.


No dependencies

Coraline has no dependencies and no Javascript code. Use Coraline with your favorite Javascript framework.


Customization

The base.scss can be used if you want to customize Coraline colors with the Sass variables. This is the list of predefined colors:

  • Brand
  • Primary
  • Secondary
  • Info
  • Warning
  • Success
  • Danger

And to customize them:

// Set your brand color
$brand: #ff7675;

// Set the contrast color,
// the default value is white
$contrast-brand: black;

// Define the rest of the colors
$primary: #6c5ce7;
$secondary: #0984e3;

$info: #74b9ff;
$contrast-info: black;

$success: #1dd1a1;
$danger: #ee5253;

$warning: #f7b731;
$contrast-warning: black;

$black: #2d3436;

// Set the font family
$font-family: Helvetica;
$font-weight: 400;

// Import Coraline scss file
@import "base.scss";
