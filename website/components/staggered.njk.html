Staggered grid view

A left to right, staggered grid built with CSS and JS.

For the staggered grid to work, you'll need to import the javascript file coraline-v0.5.3/coraline.min.js from coraline framework to the bottom of your html document. The script makes the calculation for the negative margin-top for each element, making it look staggered. It also listens for resize window events and when new elements are added to the staggered grid using a mutation observer.

All you need to do is adding the staggered class to your element. Every item you want to place inside the staggered grid, must be surrounded by an html element with the staggered-item class.
