For the staggered grid to work, you'll need to import the javascript file coraline-v0.5.1/coraline.min.js from coraline framework to the bottom of your html document. The script makes the calculation for the negative margin-top for each element, making it look staggered. It also listens for resize window events and when new elements are added to the staggered grid using a mutation observer.

All you need to do is adding the staggered class to your element. Every item you want to place inside the staggered grid, must be surrounded by an html element with the staggered-item class.

In desktop, the staggered grid will be 4 columns length by default. To overwrite this behavior, use the classes two or three for two or three columns.

In mobile, the staggered grid will be 1 column length by default. Use the classes mobile-two or mobile-three for 2 or 3 columns. The tablet will respect the desktop columns, but you may use tablet-two or tablet-three to specify a different column length if need it.
