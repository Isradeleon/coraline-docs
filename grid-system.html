Basic usage

You just need to add the grid class to your columns container. This is a 12 column system, so you may use the col-* syntax for defining the column size for each element.

If you only want same width columns, you can use the col class for each element.

Responsiveness

For defining a diferent a different column span for mobile and tablet, just use the mobile-* and tablet-* syntax with the same 12 column system.

col-4, tablet-12, mobile-6

col-4, tablet-12, mobile-6

col-4, tablet-12, mobile-12

If you don't set the tablet column span, it will respect the desktop columns. On the other hand, mobile will override them to fill the 100% width of the grid.


For large screen devices is the same, you just need to use the large-* syntax with the 12 column system. If you don't set these values, the columns you set with the col-* syntax will stay the same.

col-12, large-6
