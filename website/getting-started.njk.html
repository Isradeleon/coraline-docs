Getting started

This is a free, open source front-end component library. Its main objective is to be a fresh alternative to build high quality websites on the market.

Quick start

You may use the CDN for a quick start:

  • https://cdn.statically.io/gh/isradeleon/coraline/0.5.3/coraline-v0.5.3/coraline.min.css
  • https://cdn.statically.io/gh/isradeleon/coraline/0.5.3/coraline-v0.5.3/coraline.min.js

Easy installation

You may install the framework from npm or download the current version here. Then just import the minified CSS and JS files into your project. Here is the folder structure:

coralinecss
  • coraline-v0.5.3
    • coraline.min.css
    • coraline.min.js
  • base.scss
  • base.css
  • base.js
  • README.md
  • LICENSE

The coraline-v0.5.3/coraline.min.css and coraline-v0.5.3/coraline.min.js files are production ready. Import these files into you project and you're on the fly. Both files together are 19.3K size.


NPM

Just do npm install coralinecss from your terminal. After installing, you'll find the coraline-v0.5.3 folder with the production files. Outside this folder, you'll find the base.scss, base.css and base.js development files.


No dependencies

Coraline has no dependencies. The sidebar and staggered grid functionality are written in pure javascript. If you don't need to use these components, you may just import the coraline-v0.5.3/coraline.min.css, which is just 17K size.


Customization

The base.scss may be imported if you want to customize Coraline colors using Sass. This is the list of colors:

  • Brand
  • Primary
  • Secondary
  • Info
  • Warning
  • Success
  • Danger
// Set your brand color
$brand: #ff7675;

// Set the contrast color,
// the default value is white
$contrast-brand: black;

// Define the rest of the colors
$primary: #6c5ce7;
$secondary: #0984e3;

$info: #74b9ff;
$contrast-info: black;

$success: #1dd1a1;
$danger: #ee5253;

$warning: #f7b731;
$contrast-warning: black;

$black: #2d3436;

// Set the font family
$font-family: Helvetica;
$font-weight: 400;

// Import Coraline scss file
@import "base.scss";
